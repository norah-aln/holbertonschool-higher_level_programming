# Python - More Classes and Objects

This project explores advanced OOP concepts including class attributes, static methods, class methods, and special methods like __str__ and __repr__.

## Learning Objectives
- Understanding class vs instance attributes
- Implementing static methods and class methods
- Using __str__ and __repr__ special methods
- Working with destructors
- Dynamic attribute creation
