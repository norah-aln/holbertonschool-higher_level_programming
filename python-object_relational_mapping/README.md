
# Python - Object-relational mapping

This project demonstrates connecting Python with MySQL databases using MySQLdb and SQLAlchemy ORM.

## Tasks

**MySQLdb (Tasks 0-5)**

- Connecting to MySQL from Python

- Executing SQL queries

- Handling SQL injection

**SQLAlchemy ORM (Tasks 6-14)**

- Defining models

- CRUD operations

- Relationships between tables

## Requirements

- Python 3.8.5

- MySQLdb 2.0.3

- SQLAlchemy 1.4.22

- MySQL 8.0
