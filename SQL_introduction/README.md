
# SQL Introduction

This project contains SQL scripts demonstrating basic MySQL operations.

## Tasks

0. **0-list_databases.sql** - Lists all databases

1. **1-create_database_if_missing.sql** - Creates database hbtn_0c_0

2. **2-remove_database.sql** - Deletes database hbtn_0c_0

3. **3-list_tables.sql** - Lists all tables

4. **4-first_table.sql** - Creates table first_table

5. **5-full_table.sql** - Shows table description

6. **6-list_values.sql** - Lists all rows

7. **7-insert_value.sql** - Inserts a row

8. **8-count_89.sql** - Counts records with id=89

9. **9-full_creation.sql** - Creates and populates second_table

10. **10-top_score.sql** - Lists records by score

11. **11-best_score.sql** - Lists records with score >= 10

12. **12-no_cheating.sql** - Updates Bob's score

13. **13-change_class.sql** - Deletes records with score <= 5

14. **14-average.sql** - Computes average score

15. **15-groups.sql** - Groups records by score

16. **16-no_link.sql** - Lists records with non-NULL names

## Requirements

- MySQL 8.0

- Ubuntu 22.04 LTS

- All files end with newline

- SQL keywords in UPPERCASE

## Usage

```bash

cat <file>.sql | mysql -uroot -p [database]

```
