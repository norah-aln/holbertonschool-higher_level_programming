The ``1-my_list`` module
======================

Using ``MyList``
-------------------

First import ``MyList`` from the ``1-my_list`` module:

    >>> MyList = __import__('1-my_list').MyList

Now use it:

Test basic usage:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test with negative numbers:
    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> my_list.append(1)
    >>> my_list.append(-1)
    >>> my_list.append(0)
    >>> my_list.append(-5)
    >>> print(my_list)
    [5, 1, -1, 0, -5]
    >>> my_list.print_sorted()
    [-5, -1, 0, 1, 5]
    >>> print(my_list)
    [5, 1, -1, 0, -5]

Test with empty list:
    >>> my_list = MyList()
    >>> print(my_list)
    []
    >>> my_list.print_sorted()
    []
    >>> print(my_list)
    []

Test with one element:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> print(my_list)
    [1]
    >>> my_list.print_sorted()
    [1]
    >>> print(my_list)
    [1]

Test with two elements:
    >>> my_list = MyList()
    >>> my_list.append(2)
    >>> my_list.append(1)
    >>> print(my_list)
    [2, 1]
    >>> my_list.print_sorted()
    [1, 2]
    >>> print(my_list)
    [2, 1]

Test with already sorted list:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(4)
    >>> print(my_list)
    [1, 2, 3, 4]
    >>> my_list.print_sorted()
    [1, 2, 3, 4]
    >>> print(my_list)
    [1, 2, 3, 4]

Test with reverse sorted list:
    >>> my_list = MyList()
    >>> my_list.append(4)
    >>> my_list.append(3)
    >>> my_list.append(2)
    >>> my_list.append(1)
    >>> print(my_list)
    [4, 3, 2, 1]
    >>> my_list.print_sorted()
    [1, 2, 3, 4]
    >>> print(my_list)
    [4, 3, 2, 1]

Test with duplicate values:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> print(my_list)
    [1, 2, 1, 2]
    >>> my_list.print_sorted()
    [1, 1, 2, 2]
    >>> print(my_list)
    [1, 2, 1, 2]

Test MyList is a subclass of list:
    >>> my_list = MyList()
    >>> isinstance(my_list, list)
    True

Test MyList has print_sorted method:
    >>> my_list = MyList()
    >>> type(my_list.print_sorted)
    <class 'method'>

Test initialization with values:
    >>> my_list = MyList([3, 1, 2])
    >>> print(my_list)
    [3, 1, 2]
    >>> my_list.print_sorted()
    [1, 2, 3]

Test with large numbers:
    >>> my_list = MyList()
    >>> my_list.append(1000000)
    >>> my_list.append(1)
    >>> my_list.append(500000)
    >>> print(my_list)
    [1000000, 1, 500000]
    >>> my_list.print_sorted()
    [1, 500000, 1000000]
