# RESTful API

This project covers the fundamentals of RESTful APIs, including HTTP/HTTPS basics, API consumption, API development, security, and authentication.

## Learning Objectives
- Understanding HTTP/HTTPS protocols
- Consuming APIs with curl and Python
- Developing APIs with http.server and Flask
- Implementing API security and authentication
- Working with JSON data
