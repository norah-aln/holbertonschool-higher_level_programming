# Python - Hello, World

This project is part of the Holberton School Higher Level Programming curriculum.
It introduces the basics of Python programming, including:

- Using the Python interpreter
- Printing text and variables
- Working with strings
- Indexing and slicing
- Following Python coding style (pycodestyle)

## Files

- 2-print.py: Prints a specific sentence using print
- 3-print_number.py: Prints an integer using f-strings
- 5-print_string.py: Prints a string multiple times and slices it
- 6-concat.py: Concatenates strings
- 7-edges.py: Demonstrates string slicing
- 8-concat_edges.py: Builds a sentence using string slicing
- 9-easter_egg.py: Prints "The Zen of Python"
