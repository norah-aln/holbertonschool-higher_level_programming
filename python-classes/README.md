# Python - Classes and Objects

This project covers Object-Oriented Programming (OOP) concepts in Python including classes, objects, attributes, methods, properties, and encapsulation.

## Learning Objectives
- Understanding OOP principles
- Creating classes and objects
- Working with public, protected, and private attributes
- Implementing properties (getters and setters)
- Data encapsulation and information hiding
