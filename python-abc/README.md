
# Python - Abstract Classes and Interfaces

This project explores Object-Oriented Programming (OOP) concepts in Python including abstract base classes, interfaces, duck typing, multiple inheritance, and mixins.

## Learning Objectives

- Understanding and applying abstract classes

- Implementing interfaces using duck typing

- Extending standard base classes

- Method overriding and multiple inheritance

- Using mixins for modular behavior
